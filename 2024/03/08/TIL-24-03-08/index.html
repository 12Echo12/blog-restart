打算有时间就每天记录一下 TIL，顺手的事儿，个人认为挺有意义的！不乏一些专业知识and乱七八糟的balabala…

  • 妇女： 我国刑法对妇女的定义为年满十四周岁的女性。一直以为自己不属于妇女，原来已经是很多年啦！

  • webpack 的 chunks 分为两类：initial-chunk【入口点指定的main块即它的依赖项】 、non-initial chunk【异步块 or 按需加载动态导入的块】

  • 计数排序：遍历数组(假设最大元素为 max，最小元素为 min)，开辟额外空间记录各元素出现的次数（假设使用数组记录，那数组下标就依次对应 min ~ max 范围内的各个数），将统计的结果依次展开【需要额外开辟数组空间，不适合大范围的排序】

  • 桶排序：按照一定的策略，把元素分配到不同的桶中，然后对桶中的元素进行排序（可以选择不同的排序方法），然后再对不同的桶进行合并。

  • 基数排序（基于计数排序实现）： 先按照低位排序，然后收集；再根据高位对收集结果进行排序；重复上述操作直至到最高位，此时就是有序的！

  • github issues： github上很友好的进行代码交流的介质。haihaihai，都不好意思说自己用 github ，截至目前用处就只是 create 了几个自己的仓库存一些代码，开源的魅力我是一点儿没去体会，还有简单的静态部署功能-> github pages ，还有构建自动化工作流 -> workflow…羞愧羞愧…
