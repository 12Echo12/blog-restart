关于 JS 的执行环境: node.js VS 浏览器（插播一段）

它们两个都使用 JS 作为编程语言，主要区别如下：

  • 所做的操作不同：

    • 在浏览器中我们大多要与 DOM 进行一些交互。

    • 在 node 中我们大多要与文件、数据库等进行交互。

  • 拥有的对象不同：

    • 方便我们操作，浏览器会为我们提供document、window 等对象
    • 而 node 没有上述两个对象，取而代之的是 global 。且 node 为我们内置了许多优秀的模块，如文件系统…

  • 对模块化的支持不同：

     - 浏览器前比较广泛地支持ES Modules。但因为ES Modules是 ES6 新特性，所以有些版本较低的浏览器不支持，也需要我们用 babel 对其进行转译，向后兼容"翻译"为浏览器可以识别的语法。
 - node (从v12开始) 同时支持 CommonJS 和 ES Modules
